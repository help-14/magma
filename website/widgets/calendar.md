# Calendar Widget

The Calendar widget keeps date context visible on the dashboard.

## Best size

Default palette size: `4 x 4`.

## Settings

The Calendar widget currently has no configurable fields in the property panel.

## YAML example

```yaml
- id: calendar
  type: calendar
  title: Calendar
  x: 4
  y: 1
  w: 4
  h: 4
  config: {}
```
